# Proyecto HttpClient Petstore

Este proyecto demuestra cómo realizar una solicitud POST a la API de Petstore usando `HttpClient` en C#. El ejemplo específico crea un nuevo registro de mascota en la tienda de Test de Swagger.
El proyecto es sumamente simple y no creo que necesite explicación. Abre Program.cs y podrás verlo.

## Prerrequisitos

Antes de ejecutar este proyecto, asegúrate de tener instalado:

- [.NET SDK](https://dotnet.microsoft.com/download) (version 5.0 o superior)

## Uso

Para ejecutar el programa, usa simplemente el comando desde la línea de comandos estando en el directorio del proyecto:

```bash
dotnet run
```

Esto enviará una solicitud POST a `https://petstore.swagger.io/v2/pet` con un cuerpo de solicitud JSON que incluye detalles sobre una nueva mascota.
